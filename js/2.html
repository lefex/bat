    # 第 2 题：async/await

    ES8 增加了两个关键字 async 和 await，用来解决异步编程问题，可以使用同步的代码实现异步编程。

    (async () => {
    var a = await A();
    var b = await B(a);
    var c = await C(b);
    var d = await D(c);
})();
